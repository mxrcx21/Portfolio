Login

El login esta desarrollado en C# bajo la arquitectura MVC y utilizando pagina maestra.El logeo se hace a través de una conexion a una base de datos y odnde la contraseña esta encriptada bajo un cifrado de 128 bits al cual solo se puede desencriptar bajo un patron unico que esta dentro de una varible que esta dentro del backend.

Menu con C#

Este menu esta desarrollado en C# bajo la arquitectura MVC y con conexion a una base de datos SQL Server para realizar diferentes gestiones para un inventario. El menu tiene 2 modos de menu uno para el administrador donde se le muestra un menu con mas opciones y otra para los operarios los que se encargan realizar las gestiones principales del sistema. Todas las acciones son guardadas para que el administrador tenga un reporte de los movimientos de los empleados dentro del sistema.
